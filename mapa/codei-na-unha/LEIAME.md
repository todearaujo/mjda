# Casou onde? / codei na unha

Idiomas: [🇧🇷 Português](LEIAME.md) · [🇺🇸 English](README.md) · [🇪🇸 Español](LEEME.md)

## Objetivo comunicativo

Esta pasta preserva a versão original do mapa como museu do trabalho inicial.

## Técnica

A peça original usa HTML, CSS, JavaScript, o SVG [bruf.svg](bruf.svg) e os JSONs [porestado.json](porestado.json), [porgenero.json](porgenero.json) e [estados.json](estados.json).

## Dados

Os dados preservados cobrem 2013 a 2020, conforme a peça original, com fonte indicada como SIDRA/IBGE tabela 4406. Eles não foram alterados aqui para manter o registro histórico.

## Limitações

Esta versão mantém decisões e limitações originais: dados antigos, autoplay simples e cálculo/documentação menos explícitos.

## Como rodar

Na raiz do repositório, rode `python3 -m http.server 8000` e acesse `http://localhost:8000/mapa/codei-na-unha/`.
